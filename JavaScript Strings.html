JavaScript 문자열은 텍스트를 저장하고 조작하는데 사용됩니다.

JavaScript Strings

문자열은 단순히 "John Doe"과 같은 일련의 문자들을 저장합니다.

문자열은 인용 부호 안에 어떤 텍스트도 될 수 있습니다. 단일 또는 이중 따옴표를 사용할 수 있습니다 :

Example

var carname = "Volvo XC60";
var carname = 'Volvo XC60';
Try it Yourself »

문자열을 둘러싼 인용 부호와 일치하지 않는 따옴표를 문자열 안에서 사용할 수 있습니다 :

Example

var answer = "It's alright";
var answer = "He is called 'Johnny'";
var answer = 'He is called "Johnny"';
Try it Yourself »

String Length

문자열(String 객체)의 길이는 내장된 속성(property) length 에서 알 수 있다 :

Example

var txt = "ABCDEFGHIJKLMNOPQRSTUVWXYZ";
var sln = txt.length;
Try it Yourself »

특수 문자(Special Characters)

문자열은 인용부호 안에 쓰여야 하기 때문에,  JavaScript 는 다음과 같은 문자열을 잘못 이해할 것이다:

var y = "We are the so-called "Vikings" from the north."

문자열은  "We are the so-called "로 잘릴 것이다.

이 문제를 피하기 위한 해결책은,  \  escape 문자를 사용하는 것이다.

백슬래시(backslash) escape 문자 는 특수문자(special character)를 문자열 문자로 바꾸어준다:

Example

var x = 'It\'s alright';
var y = "We are the so-called \"Vikings\" from the north."
Try it Yourself »

escape 문자 (\) 는 문자열에 다른 특수 문자들을 넣는데도 사용될 수 있다.

다음은 백슬래시(backslash) 기호로 텍스트 문자열에 추가 할 수 있는 다른 특수 문자들 목록이다 :

Code Outputs
\' single quote
\" double quote
\\ backslash
\n new line
\r carriage return
\t tab
\b backspace
\f form feed

긴 코드 줄을 나누기

최적의 가독성을 위해서, 프로그래머들은 종종 80자 보다 긴 코드 줄을 피하기를 좋아한다. 

JavaScript 문이 한 줄에 들어가지 않으면, 나누기 가장 좋은 곳은 연산자(operator) 다음이다:

Example

document.getElementById("demo").innerHTML =
"Hello Dolly.";
Try it Yourself »

또는, 백슬래시(backslash)를 가지고 텍스트 문자열 안에서 코드 줄을 나눌 수도 있다.

Example

document.getElementById("demo").innerHTML = "Hello \
Dolly!";
Try it Yourself »
긴 문자열을 나눌 가장 안전한(약간 느릴지라도) 방법은 문자열 더하기를 사용하는 것이다:

Example

document.getElementById("demo").innerHTML = "Hello" +
"Dolly!";
Try it Yourself »
 백슬래시(backslash)를 가지고 코드 줄(code line)을 나눌 수 는 없다 !! :

Example

document.getElementById("demo").innerHTML = \
"Hello Dolly!";
Try it Yourself »

문자열은 객체도 될 수 있다

일반적으로는, JavaScript 문자열은 var firstName = "John"  처럼 문자적으로 생성된 기본 값(primitive value)일 수 있습니다.

그러나,  var firstName = new String("John") 처럼  new keyword 를 사용하여 객체로 생성 할 수 도 있습니다.

Example

var x = "John";
var y = new String("John");

// typeof x will return string
// typeof y will return object
Try it Yourself »
  문자열을 객체로 생성하지 마시오. 실행 속도가 느려지며, 바람직하지 않은 부작용이 발생합니다.

String 속성 및 메서드

"John Doe"와 같은 원시 값(primitive values)은 (객체가 아니기 때문에) 속성과 메서드를 가질 수 없다. 

하지만 JavaScript 에서는 속성과 메서드가 실행될 때는 원시 값(primitive value)을 객체처럼 취급하기 때문에 원시 값(primitive value)에도 속성과 메서드가 사용가능하다.

문자열에서 문자열 찾기(finding)

indexOf() 메서드는 문자열 안에서 지정된 텍스트가 처음으로 발견되는 index(위치)를 반환한다:

Example

var str = "Please locate where 'locate' occurs!";
var pos = str.indexOf("locate");
Try it Yourself »
lastIndexOf() 메서드는 문자열 안에서 지정된 텍스트가 마지막으로 발견되는 index(위치)를 반환한다

Example

var str = "Please locate where 'locate' occurs!";
var pos = str.lastIndexOf("locate");
Try it Yourself »

지정된 텍스트를 찾을 수없는 경우, indexOf() 메소드와 lastIndexOf() 메서드 모두  -1 을 반환한다 .

문자열에서 문자열 탐색(searching)

search() 메서드는 문자열 안에서 지정된 텍스트를 탐색하여 일치하는 곳의 위치를 반환한다:

Example

var str = "Please locate where 'locate' occurs!";
var pos = str.search("locate");
Try it Yourself »
  눈치 챘습니까?
  indexOf() 와 search() 두 메서드는 같습니다.
  둘 다 같은 매개변수를 받아서 같은 같을 반환합니다.
  두 메서드는 같습니다, 그러나 search() 메서드는 더 강력한  탐색 값을 취할 수 있습니다.
  강력한 탐색 값에 대하여서는  regular expression 에 관한  장에서 배울 것입니다.

문자열의 부분 추출

문자열의 일부분을 추출하는데는 세가지 메서드가 있다:
  • slice(start,end)
  • substring(start,end)
  • substr(start,length)

slice() method

slice()는 문자열의 일부를 추출하여 추출된 부분을 새로운 문자열로 반환한다.
메서드는 2 매개변수: 시작 인덱스(위치) 와 끝 인덱스(위치)를 가진다.
다음 예는 문자열의 위치 7 부터 13 까지의 부분을 잘라낸다.

Example

var str = "Apple, Banana, Kiwi";
var res = str.slice(7,13);

The result of res will be:

Banana
Try it Yourself »
파라메터가 음수이면, 위치는 문자열의 끝에서 부터 계수된다.
다음 예는 문자열의 위치 -12 부터 -6 까지의 부분을 잘라낸다.

Example

var str = "Apple, Banana, Kiwi";
var res = str.slice(-12,-6);

The result of res will be:

Banana
Try it Yourself »
두번 째 파라메터를 생략하면, 메서드는 문자열의 나머지 모두를 잘라낸다.

Example

var res = str.slice(7);
Try it Yourself »
또는, 끝에서 부터 계수하여 ...

Example

var res = str.slice(-12);
Try it Yourself »

substring() method

substring()은 slice()와 유사하다.
다른 점은 substring()은 음수 인덱스를 허용하지 않는 것이다.

Example

var str = "Apple, Banana, Kiwi";
var res = str.substring(7,13);

The result of res will be:

Banana
Try it Yourself »
두 번째 파라메터를 생략하면, substring()은 문자열의 나머지 모두를 잘라낸다.

substr() method

substr()은 slice()와 유사하다.
다른 점은 두 번째 파라매터가 추출될 부분의 길이(length)를 지정한 다는 것이다.

Example

var str = "Apple, Banana, Kiwi";
var res = str.substr(7,6);

The result of res will be:

Banana
Try it Yourself »
첫 번째 파라메터가 음수이면, 위치는 문자열의 끝부터 계수된다.
두 번째 파라메터는 길이를 정의하므로 음수 일 수 없다.
두 번째 파라메타를 생략하면, substr()는 문자열의 나머지 모두를 잘라낸다.

문자열 내용 교채(replacing)

replace() 메서드는 문자열에서 특정 값을 다른 값으로 대체합니다.

Example

str = "Please visit Microsoft!";
var n = str.replace("Microsoft","W3Schools");
Try it Yourself »
replace() 메서드 또한  탐색 값으로 regular expression을 취할 수 있다.

Upper Case and Lower Case

toUpperCase() / toLowerCase() 메서드를 사용하여 문자열을 대문자/소문자로 변환할 수 있다.:

Example

var text1 = "Hello World!";       // String
 var text2 = text1.toUpperCase();  // text2 is text1 converted to upper
Try it Yourself »

Example

var text1 = "Hello World!";       // String
 var text2 = text1.toLowerCase();  // text2 is text1 converted to lower
Try it Yourself »

concat() method

concat() 은 둘 이상의 문자열을 연결해준다.

Example

var text1 = "Hello";
var text2 = "World";
text3 = text1.concat(" ",text2);
Try it Yourself »
concat() method 는 + 연산자 대신에 사용될 수 있다. 다음의 두 코드 줄은 같다:

Example

var text = "Hello" + " " + "World!";
var text = "Hello".concat(" ","World!");

문자열에서 문자 추출

문자열의 문잘르 추출하는  두 가지 method 가 있다.
  • charAt(position)
  • charCodeAt(position)

charAt() method

charAt() method 는 문자열에서 지정한 인덱스(위치)의 문자를 반환한다.

Example

var str = "HELLO WORLD";
str.charAt(0);            // returns H
Try it Yourself »

charCodeAt() method

charCodeAt() method 는 문자열에서 지정한 인덱스(위치)의 문자의 unicode를 반환한다.

Example

var str = "HELLO WORLD";

str.charCodeAt(0);         // returns 72
Try it Yourself »

문자열을 배열로 변환

문자열은 split() 메서드를 사용하여 배열로 변환될 수 있다. :

Example

var txt = "a,b,c,d,e";   // String
 txt.split(",");          // Split on commas
 txt.split(" ");          // Split on spaces
 txt.split("|");          // Split on pipe
Try it Yourself »
구분자(separator)가 생략되면, 반환된 배열은 인덱스 [0]에 전체 문자열을 갖을 것이다.
구분자가 "" 이면, 반환된 배열은 단일 문자들의 배열이 된다.

Example

var txt = "Hello";       // String
 txt.split("");           // Split in characters
Try it Yourself »
