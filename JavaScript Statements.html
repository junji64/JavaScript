JavaScript 프로그램(programs)

컴퓨터 프로그램은 컴퓨터에서 "실행"해야 할 "명령"목록입니다.

프로그래밍 언어에서 이러한 프로그래밍 명령어를 명령문(statements)이라고합니다.

JavaScript 프로그램은 프로그래밍 명령문(statements)의 목록입니다.

Example

var x = 5;
var y = 6;
var z = x + y;
document.getElementById("demo").innerHTML = z;
Try it Yourself »
HTML에서 JavaScript 프로그램은 웹 브라우저에 의해 실행됩니다.

JavaScript 명령문(Statements)

JavaScript 문장은 다음과 같이 구성됩니다.

값, 연산자, 표현식(expression), 키워드 및 주석.

이 JavaScript 실행문은 부라우저에게 id="demo" 인 HTML 요소 안에 "Hello Dolly" 를 쓰라고 말한다.:

Example

document.getElementById("demo").innerHTML = "Hello Dolly.";
Try it Yourself »

Semicolon ;

세미콜론은 JavaScript 문을 분리한다.

일반적으로 각 실행문 끝에 세미콜론을 추가합니다.

a = 5;
b = 6;
c = a + b;
Try it Yourself »

세미콜론을 사용하면 한 줄에 여러 문장을 작성할 수 있게 해 줍니다.

a = 5; b = 6; c = a + b;
Try it Yourself »
   웹에서, 세미콜론이 없는 예제도 볼 수 있을 겁니다. 마지막 문장에서 세미콜론은 JavaScript 에서 선택 사항이기 때문이나, 절대로 권장합니다.

JavaScript 공백(White Space)

JavaScript 는 여분의 공백을 무시합니다. 더 쉽게 읽혀지도록 스크립트에 공백을 추가 할 수 있습니다.

다음의 두 줄은 동일 합니다.:

var person = "Hege";
var person="Hege";

연산자 ( = + - *  /  ) 들 주위에는 공백을 넣는 것이 바람직합니다.

var x = y + z;

JavaScript Line 길이와 Line 자르기

읽기에 수월하도록 프로그래머들은 종종 코드 라인의 길이가 80자 이내가 되도록 한다.

만약, JavaScript 문이 한 줄에 들어 가지 않으면, 자르기 가장 좋은 곳은 연산자 다음이다: 

Example

document.getElementById("demo").innerHTML =
"Hello Dolly.";
Try it Yourself »

JavaScript Code Blocks

JavaScript 문은 중괄호( {...} ) 안에,  코드 블록으로 함께 그룹화 할 수 있습니다.

코드 블록의 목적은 함께 실행될 문장들을 함께 선언하는 것입니다.

블록으로 그룹화하는 문들의 좋은 예는, JavaScript 함수(functions)  에 서 이다.:

Example

function myFunction() {
    document.getElementById("demo").innerHTML = "Hello Dolly.";
    document.getElementById("myDIV").innerHTML = "How are you?";
}

JavaScript 키워드(Keywords)

JavaScript 문들은 실행될 JavaScript 행동을 식별하기 위해서 종종 키워드(keyword)로 시작된다.

Keyword Description
break Terminates a switch or a loop
continue Jumps out of a loop and starts at the top
debugger Stops the execution of JavaScript, and calls (if available) the debugging function
do ... while Executes a block of statements, and repeats the block, while a condition is true
for Marks a block of statements to be executed, as long as a condition is true
function Declares a function
if ... else Marks a block of statements to be executed, depending on a condition
return Exits a function
switch Marks a block of statements to be executed, depending on different cases
try ... catch Implements error handling to a block of statements
var Declares a variable

    JavaScript 키워드(keyword)는 예약어이다. 예약어는 변수 명으로 사용될 수 없다.
