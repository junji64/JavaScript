JavaScript Errors - Throw 와 Try to Catch

try 문은 코드블록의 오류를 테스트할 수 있다.

catch 문은 오류를 처리하게 해준다.

throw 문은 사용자 지정 오류를 생성할 수 있다.

Errors Will Happen!

JavaScript 엔진이 JavaScript 코드를 실행하면, 여러가지 오류가 발생 할 수 있다:

그것은 문법 오류 일 수 있으며, 일반적인 코딩 오류거나 오타 일 수 있다.

잘못 썼거나 언어에서의 빠진 기능 (아마도 브라우저 차이 때문).

사용자 또는 인터넷 서버로부터의  잘못된 입력에 의한 오류 일 수 도 있다.

그리고, 물론 많은 다른 예상할 수 없는 것들 일 수 있다.

JavaScript Throws Errors

무엇인가가 잘 못 되서 오류가 발생하면, JavaScript 엔진은 일반적으로 중지하고, 오류 메시지를 생성합니다.

이것의 기술적인 용어는 : JavaScript 가 error를 throw 했다.

JavaScript try and catch

try 문은 코드블록이 실행 되는 동안 오류에 대하여 테스트되도록 정의 할 수 있다.

catch 문은, try 블록에서 오류가 발생하면, 실행될 코드 블록을 정의 할 수 있게 해 준다.

JavaScript 문에서  try 와 catch 는 쌍으로 따라온다.

Syntax

try {
    Block of code to try
}
catch(err) {
    Block of code to handle errors
}

Examples

아래의 예는 의도적으로 try 블록 안의 코드에 오타를 만들어 넣었다.

catch 블록은 try 블록에서 오류를 캐치하고 그것을 처리하는 코드를 실행한다.:

Example

<!DOCTYPE html>
<html>
<body>

<p id="demo"></p>

<script>
try {
    adddlert("Welcome guest!");
}
catch(err) {
    document.getElementById("demo").innerHTML = err.message;
}
</script>

</body>
</html>

Throw 문

throw 문은 사용자 지정 오류를 생성할 수 있게 해준다.

올바른 기술적 용어는 예외를 만들거나 던지다(create or throw an exception) 이다.

throw 문을 try 와 catch 와 함께 사용하면, 프로그램 흐름을 제어 할 수 있고 사용자 지정 오류 메시지를 생성할 수 있다.

Syntax

throw "Too big";    // throw a text
 throw 500;          // throw a number

예외(exception)는 JavaScript String, Number, Boolean 또는 Object 일 수 있다.

Example

이 예는 입력 변수 값을 검사한다. 값이 잘못 되면 예외(exception(error))가 던져진다(is thrown). 에러는 catch 문에 의해서 잡혀(caught)지고 사용자 지정 오류 메시지가 보여진다:

<!DOCTYPE html>
<html>
<body>

<p>Please input a number between 5 and 10:</p>

<input id="demo" type="text">
<button type="button" onclick="myFunction()">Test Input</button>
<p id="message"></p>

<script>
function myFunction() {
    var message, x;
    message = document.getElementById("message");
    message.innerHTML = "";
    x = document.getElementById("demo").value;
    try {
        if(x == "") throw "empty";
        if(isNaN(x)) throw "not a number";
        x = Number(x);
        if(x < 5) throw "too low";
        if(x > 10) throw "too high";
    }
    catch(err) {
        message.innerHTML = "Input is " + err;
    }
}
</script>

</body>
</html>
