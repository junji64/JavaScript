JavaScript 데이터 타입

JavaScript 변수는 다양한 데이터 타입을 담을 수 있다: 숫자, 문자열, 배열, 객체 등. :

var length = 16;                               // Number
 var lastName = "Johnson";                      // String
 var cars = ["Saab", "Volvo", "BMW"];           // Array
var x = {firstName:"John", lastName:"Doe"};    // Object

데이터 타입의 개념

프로그래밍에서 데이터 타입은 중요한 개념이다.

변수에 연산을 적용하기 위해서는, 타입에 대하여 아는 것이 중요하다.

데이터 타입 없이, 컴퓨터는 다음을 해결할 수 없다:

var x = 16 + "Volvo";

"Volvo"에 16을 더하는 것이 의미있는가? 결과가 있는가? 에러를 발생시키는가?

JavaScript는 위의 것을 다음과 같이 간주할 것이다:

var x = 16 + "Volvo";

  만약에 두 번째 피연산자가 문자열이면,  JavaScript 는 첫 번째 피 연산자를 문자열로 처리한다.

Example

var x = 16 + "Volvo";
Try it Yourself »
JavaScript는 수식을 왼쪽에서 오른쪽으로 평가한다. 다른 서열은 다른 결과를 만든다.

JavaScript:

var x = 16 + 4 + "Volvo";

Result:

20Volvo
Try it Yourself »

JavaScript:

var x = "Volvo" + 16 + 4;

Result:

Volvo164
Try it Yourself »
첫 번째 예에서는,  JavaScript는 16과 4를 ,"Volvo"를 만나기 전 까지는, 숫자로 다룬다.
두 번째 예에서, 첫 번째 피 연산자가 문자열이므로, 모든 피 연산자는 문자열로 간주 된다.

JavaScript 는 동적형식(Dynamic Types)을 가진다

JavaScript 는 동적 형식(dynamic types)을 가진다. 이는 같은 변수가 다른 형식으로 사용될 수 있다는 것을 의미한다 :

Example

var x;               // Now x is undefined
var x = 5;           // Now x is a Number
 var x = "John";      // Now x is a String

JavaScript 문자열(Strings)

문자열(string)은  "John Doe" 와 같은 일련의 문자를 저장하는 변수입니다.

문자열은 인용 부호 안에 어떤 텍스트도 올 수 있습니다.  단일 또는 이중 따옴표를 사용할 수 있습니다 :

Example

var carName = "Volvo XC60";   // Using double quotes
 var carName = 'Volvo XC60';   // Using single quotes

문자열을 둘러싼 인용 부호와 일치하지 않는 따옴표를 문자열 안에 사용할 수 있습니다 :

Example

var answer = "It's alright";             // Single quote inside double quotes
 var answer = "He is called 'Johnny'";    // Single quotes inside double quotes
 var answer = 'He is called "Johnny"';    // Double quotes inside single quotes
Try it yourself »

JavaScript Numbers

JavaScript 는 숫자에 하나의 형식만 있습니다. 숫자는 소수점(decimals)과 함께, 또는 없이 쓸 수 있다 :

Example

var x1 = 34.00;     // Written with decimals
 var x2 = 34;        // Written without decimals

특별히 크거나 작은 숫자는 과학적(지수:exponential) 표기법으로 쓸 수 있다 :

Example

var y = 123e5;      // 12300000
 var z = 123e-5;     // 0.00123
Try it yourself »

JavaScript Booleans

부울(Booleans)은 true 또는 false 두 개의 값을 가질 수 있습니다.:

Example

var x = true;
var y = false;

부울은 종종 조건부 테스트에 사용됩니다.

JavaScript 배열(Arrays)

JavaScript 배열은 각진괄호(square brackets)로 쓴다.

배열의 항목들은 쉼표(comma)로 구분된다.

다음은 세개의 항목(자동차 이름)을 갖는 cars 라는 이름의 배열을 선언(생성)한다.

Example

var cars = ["Saab", "Volvo", "BMW"];
Try it Yourself »

배열 인덱스는 0부터 시작하며, 첫 번째 항목 [0], 두 번째는 [1], 등 이 됨을 의미한다.

JavaScript 객체(Objects)

JavaScript 객체는 중괄호(curly braces)로 쓰입니다.

중괄호 안에 객체의 속성(property)은 쉼표로 구분된 이름:값(name : value)의 쌍으로 정의된다 .

Example

var person = {firstName:"John", lastName:"Doe", age:50, eyeColor:"blue"};
Try it Yourself »

위의 예에서 객체(person)은 4가지 성질(property)를 가진다: firstName, lastName, age, eyeColor.

typeof  연산자

JavaScript 변수의 형식을 찾기 위해  typeof 연산자를 사용할 수 있습니다.

Example

typeof "John"                // Returns string
typeof 3.14                  // Returns number
 typeof false                 // Returns boolean
 typeof [1,2,3,4]             // Returns object
 typeof {name:'John', age:34} // Returns object
Try it Yourself »
  JavaScript 에서, 배열은 객체의 특별한 형태이다. 따라서 typeof [1,2,3,4]는 객체를 반환한다. 

Undefined

JavaScript 에서 값이 없는 변수는 undefined 값 을 가진다. typeof 또한 undefined 이 다.

Example

var person;                  // Value is undefined, type is undefined
Try it Yourself »
어떤 변수도 값을 undefined 로 설정하여 비어 있을 수 있다. 형식 또한 undefined 가 된다.

Example

person = undefined;          // Value is undefined, type is undefined
Try it Yourself »

빈 값(empty value)

빈 값은 undefined 와는 아무 관련이 없다.

빈 문자열 변수는 값과 type을 가진다..

Example

var car = "";                // The value is "", the typeof is string
Try it Yourself »

Null

JavaScript 에서 null 은 "아무것도 아님(nothing)"이다.

이는 존재 하지 않는 무언가를 상징한다.

불행히도, JavaScript에서, null 의 데이타형식은 객체(object)이다.

  JavaScript 에서, 배열은 객체의 특별한 형태이다. 따라서 typeof [1,2,3,4]는 객체를 반환한다. 

객체를 null 로 설정하여 비울 수 있다.

Example

var person = null;           // Value is null, but type is still an object
Try it Yourself »

또는 객체를 undefined로 설정하여 비울 수 도 있다.

Example

var person = undefined;     // Value is undefined, type is undefined
Try it Yourself »

undefined 와 null 의 차이

typeof undefined             // undefined
typeof null                  // object
null === undefined           // false
null == undefined            // true
Try it Yourself »
