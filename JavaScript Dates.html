Creating Date Objects

Date 객체는 날짜(dates) 가지고 일할 수 있게 해준다.

날짜(date)는 a year, a month, a day, an hour, a minute, a second, 과 milliseconds 로 구성된다.

Date 객체는 new Date() 생성자(constructor)로 생성된다..

날짜(date)를 시작하는 4 가지 방법:

new Date()
new Date(milliseconds)
new Date(date String)
new Date(year, month, day, hours, minutes, seconds, milliseconds)

new Date() 를 사용하여, 현재 날짜와 시간(current date and time)으로 새로운 날짜 객체가 만들어진다:

Example

<script>
var d = new Date();
document.getElementById("demo").innerHTML = d;
</script>
Try it Yourself »

new Date(date string)를 사용하여, 특정 날짜와 시간(specified date and time)을 가진 새로운 날짜 객체가 만들어진다.:

Example

<script>
var d = new Date("March 27, 2020 14:45:00");
document.getElementById("demo").innerHTML = d;
</script>
Try it Yourself »
new Date(number)를 사용하여, zero time 에 number를 더한 시간을 가진 새로운 날짜 객체가 만들어진다

new Date(0)은 01 January 1970 00:00:00 UTC. 숫자는milliseconds 로 주어진다:

1970 년 1 월 1 일에서 1 천억 밀리 초를 뺀 값은 대략 1966 년 10 월 31 일입니다.

Example

<script>
var d = new Date(1585287988641);
document.getElementById("demo").innerHTML = d;
</script>
Try it Yourself »

  milliseconds 로 하루는 86,400,000 millisecond 이다.

new Date(7 numbers)를 사용하여, 특정 날짜와 시간(specified date and time)을 가진 새로운 날짜 객체가 만들어진다.

7 numbers 는 the year, month, day, hour, minute, second, 과 millisecond 를 순서대로 나타낸다.:

Example

<script>
var d = new Date(2020,2,27,14,33,30,0);
document.getElementById("demo").innerHTML = d;
</script>
Try it Yourself »

위의 예에서, 마지막 4~5 parameters는 생략이 가능하다:

Example

<script>
var d = new Date(2020,2,27);
document.getElementById("demo").innerHTML = d;
</script>
Try it Yourself »

JavaScript 는 월(months)을 0 부터 11까지로 한다. January 는 0. December 는s 11.

월(Month)은 생략 할 수 없습니다. 하나의 매개변수만 제공하면 milli-seconds로 처리됩니다.

Example

<script>
var d = new Date(2020);
document.getElementById("demo").innerHTML = d;
</script>
Try it Yourself »

Date Methods

Date 객체가 생성되면, 많은 methods 들을 작동할 수 있다.

Date 메서드들은 당신이 로컬 시간 또는 UTC (universal, or GMT) 시간을 사용하여 객체의 year, month, day, hour, minute, second, and millisecond 들을 얻거나 설정할 수 있다.

 Date methods are covered in a later chapter.

Dates 표시

HTML에 date 객체를 표시하면, toString() 메서드로 자동적으로 문자열로 변환된다. 

Example

<p id="demo"></p>

<script>
d = new Date();
document.getElementById("demo").innerHTML = d;
</script>

Is the same as:

<p id="demo"></p>

<script>
d = new Date();
document.getElementById("demo").innerHTML = d.toString();
</script>
Try it Yourself »

toUTCString() method는 date를 UTC 문자열(date 표시 표준)로 변환한다.

Example

<script>
var d = new Date();
document.getElementById("demo").innerHTML = d.toUTCString();
</script>
Try it Yourself »

toDateString() method는 date를 좀더 읽기 쉬운 형식으로 변환한다.:

Example

<script>
var d = new Date();
document.getElementById("demo").innerHTML = d.toDateString();
</script>
Try it Yourself »

  Date 객체는 정적(static)이지, 동적(dynamic)이지 않다. 컴퓨터 타임을 시간에 따라 변하지만 date 객체는 한번 생성되면 변화하지 않는다.

JavaScript Date Formats

유효한 JavaScript Date 문자열

세 가지 유효한 JavaScript date 포멧이 있다:

Type Example
ISO Date "2020-03-25" (The International Standard)
Short Date "03/25/2020"
Long Date "Mar 25 2020" or "25 Mar 2020"

JavaScript ISO Dates

ISO 8601 syntax (YYYY-MM-DD)는 최신의(선호되는) JavaScript date 포맷이다:

Example (Complete date)

var d = new Date("2020-03-25");
Try it Yourself »

Or as: "2020-12" (YYYY-MM):

Example (Year and month)

var d = new Date("2020-03");
Try it Yourself »

Or as "2020" (YYYY):

Example (Only year)

var d = new Date("2020");
Try it Yourself »

Or as "2020-12-24T12:00:00":

Example (Complete date plus hours, minutes, and seconds)

var d = new Date("2020-03-25T12:00:00");
Try it Yourself »

 date 문자열에서 T(date 와 time 사이)는 UTC time 임을 나타낸다.

  UTC (Universal Time Coordinated)  는 GMT (Greenwich Mean Time) 과 같다.

Example

Example

Short dates 은 다음과 같이 "MM/DD/YYYY" 구문으로 사용한다.:

Example

어떤 브라우져에서는 "-" 를 구분자로 쓸 수 있다.:

JavaScript 는 "YYYY/MM/DD" 도 허용한다.:

Example

var d = new Date("2020/03/25");

JavaScript Date Methods

Date methods 는 date 값들 (years, months, days, hours, minutes, seconds, milliseconds)을 얻거나 설정하도록 해준다. 

Date Get Methods

Get methods 들은 date의 일부분을 얻는데 사용된다. :

Method Description
getDate() Get the day as a number (1-31)
getDay() Get the weekday as a number (0-6)
getFullYear() Get the four digit year (yyyy)
getHours() Get the hour (0-23)
getMilliseconds() Get the milliseconds (0-999)
getMinutes() Get the minutes (0-59)
getMonth() Get the month (0-11)
getSeconds() Get the seconds (0-59)
getTime() Get the time (milliseconds since January 1, 1970)

getTime() Method

getTime() 는 01.01.1970 로부터 milliseconds 수를 반환한다.:

Example

<script>
var d = new Date();
document.getElementById("demo").innerHTML = d.getTime();
</script>
Try it Yourself »

getFullYear() Method

getFullYear() 는 date의 year를 네자리 숫자로 반환한다.:

Example

<script>
var d = new Date();
document.getElementById("demo").innerHTML = d.getFullYear();
</script>
Try it Yourself »

getDay() Method

getDay() 는 요일(weekday) 을 숫자 (0-6)로 반환한다.:

Example

<script>
var d = new Date();
document.getElementById("demo").innerHTML = d.getDay();
</script>
Try it Yourself »

당신은 이름 배열과 getDay() 를 사용하여 요일(weekday)을 이름으로 얻어낼 수 있다:

Example

<script>
var d = new Date();
var days = ["Sunday","Monday","Tuesday","Wednesday","Thursday","Friday","Saturday"];
document.getElementById("demo").innerHTML = days[d.getDay()];
</script>
Try it Yourself »

Date Set Methods

Set methods 는 date의 일 부분을 설정하는데 사용한다.:

Method Description
setDate() Set the day as a number (1-31)
setFullYear() Set the year (optionally month and day)
setHours() Set the hour (0-23)
setMilliseconds() Set the milliseconds (0-999)
setMinutes() Set the minutes (0-59)
setMonth() Set the month (0-11)
setSeconds() Set the seconds (0-59)
setTime() Set the time (milliseconds since January 1, 1970)

setFullYear() Method

setFullYear() 는 date 객체를 특정 date 로 설정한다.eg)  January 14, 2020:

Example

<script>
var d = new Date();
d.setFullYear(2025, 0, 14);
document.getElementById("demo").innerHTML = d;
</script>
Try it Yourself »

setDate() Method

setDate() 는 월(month)의 일을 (1-31) 설정한다:

Example

<script>
var d = new Date();
d.setDate(20);
document.getElementById("demo").innerHTML = d;
</script>
Try it Yourself »

setDate() 는 date 에 일들(days)을 더하는데 사용될 수 도 있다.:

Example

<script>
var d = new Date();
d.setDate(d.getDate() + 50);
document.getElementById("demo").innerHTML = d;
</script>
Try it Yourself »

  일들을 더하면, 월과 년의 이동이 date 객체에 의해서 자동적으로 처리된다.

Date 입력 - Dates 구문 분석

유효한 date 문자열을 가지고 있다면, 이를 milliseconds 로 변환하는데 Date.parse() method 를 사용할 수 있다. 

Date.parse() 는 date 와  January 1, 1970 사이의 milliseconds 수를 반환한다:

Example

<script>
var msec = Date.parse("March 21, 2012");
document.getElementById("demo").innerHTML = msec;
</script>
Try it Yourself »

그 다음에 number of milliseconds 를 date object 로 변환할 수 있다:

Example

<script>
var msec = Date.parse("March 21, 2012");
var d = new Date(msec);
document.getElementById("demo").innerHTML = d;
</script>
Try it Yourself »

Dates 비교

Dates 는 쉽게 비교될 수 있다.

다음 예는오늘 date 를 January 14, 2100 과 비교한다:

Example

var today, someday, text;
today = new Date();
someday = new Date();
someday.setFullYear(2100, 0, 14);

if (someday > today) {
    text = "Today is before January 14, 2100.";
} else {
    text = "Today is after January 14, 2100.";
}
document.getElementById("demo").innerHTML = text;
Try it Yourself »

   JavaScript 는 월을 0 부터 11 까지로 한다. January is 0. December is 11.
