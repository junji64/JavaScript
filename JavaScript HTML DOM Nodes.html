노드 (HTML Elements) 추가 및 제거

새로운 HTML Elements (Nodes) 만들기

HTML DOM 에 새 요소를 추가하려면 먼저 요소(요소 노드)를 생성해야하고, 다음은 기존 요소에 덧붙입니다(append).

 Example

<div id="div1">
<p id="p1">This is a paragraph.</p>
<p id="p2">This is another paragraph.</p>
</div>

<script>
var para = document.createElement("p");
var node = document.createTextNode("This is new.");
para.appendChild(node);

var element = document.getElementById("div1");
element.appendChild(para);
</script>
Try it Yourself »

Example Explained 

이 코드는 새로운 <p> 요소를 만듭니다:

var para = document.createElement("p");

<p> 요소에 텍스트를 추가하려면, 먼저 텍스트 노드를 생성해야 합니다. 이 코드는 텍스트 노드를 생성합니다 :

var node = document.createTextNode("This is a new paragraph.");

그 다음 <p> 요소에 텍스트 노드를 추가해야 합니다 :

para.appendChild(node);

마지막으로 기존 요소에 새로운 요소를 추가해야합니다.

이 코드는 기존 요소를 찾습니다:

var element = document.getElementById("div1");

이 코드는 기존의 요소에 새로운 요소를 추가합니다 :

element.appendChild(para);

새로운 HTML 요소 만들기 - insertBefore()

앞의 예에서 appendChild() method 는 부모의 마지막 자식으로 새 요소를 추가 합니다.

그것을 원하지 않는다면 insertBefore() method 를 사용할 수 있습니다.:

Example

<div id="div1">
<p id="p1">This is a paragraph.</p>
<p id="p2">This is another paragraph.</p>
</div>

<script>
var para = document.createElement("p");
var node = document.createTextNode("This is new.");
para.appendChild(node);

var element = document.getElementById("div1");
var child = document.getElementById("p1");
element.insertBefore(para,child);
</script>

기존의 HTML 요소 제거

HTML 요소를 제거하려면 요소의 부모를 알고 있어야 합니다 :

Example

<div id="div1">
<p id="p1">This is a paragraph.</p>
<p id="p2">This is another paragraph.</p>
</div>

<script>
var parent = document.getElementById("div1");
var child = document.getElementById("p1");
parent.removeChild(child);
</script>
Try it Yourself »

Example Explained 

이 HTML 문서는 두 개의 자식 노드 (두 <p> 요소) 를 갖는 <div> 요소를 갖습니다.:

<div id="div1">
<p id="p1">This is a paragraph.</p>
<p id="p2">This is another paragraph.</p>
</div>

id="div1" 를 가진 요소를 찾습니다:

var parent = document.getElementById("div1");

id="p1" 를 가진 <p> 요소를 찾습니다::

var child = document.getElementById("p1");

부모로부터 자식을 제거합니다:

parent.removeChild(child);

  부모를 참조하지 않고 요소를 제거 할 수있다면 좋을 것이다. 하지만. DOM은 제거 할 요소 및   그의 부모 모두를 알 필요가 있다.

다음은 일반적인 해결 방법입니다 : 제거 할 자식를 찾아 부모를 찾기 위해서 parentNode 속성을 사용합니다 

var child = document.getElementById("p1");
child.parentNode.removeChild(child);

HTML 요소 교체

HTML DOM에 요소를 교체하려면, replaceChild() 메서드를 사용 :

Example

<div id="div1">
<p id="p1">This is a paragraph.</p>
<p id="p2">This is another paragraph.</p>
</div>

<script>
var para = document.createElement("p");
var node = document.createTextNode("This is new.");
para.appendChild(node);

var parent = document.getElementById("div1");
var child = document.getElementById("p1");
parent.replaceChild(para,child);
</script>

JavaScript HTML DOM 노드목록(Nodelist)

노드목록(nodelist)은 노드의 배열 (모든 HTML 요소의 배열과 같이) 이다.

HTML DOM Node List

getElementsByTagName() method 는 노드목록(node list) 을 반환한다. 노드 목록은 노드의 배열이다.

다음 코드는 문서에 있는 모든 <p> 노드를 선택합니다:

Example

var x = document.getElementsByTagName("p");

The nodes can be accessed by an index number. To access the second <p> node you can write:

y = x[1];
Try it Yourself »

Note: The index starts at 0.

HTML DOM Node List Length

length 속성은 노드 목록에 있는 노드의 수를 정의합니다.

Example

var myNodelist = document.getElementsByTagName("p");
document.getElementById("demo").innerHTML = myNodelist.length;
Try it Yourself »

Example explained:

  1. 노드 목록(배열)에 모든 <p> 요소를 가져온다
  2. 노드 목록의 길이를 표시
length 속성은 노드 목록의 노드들에 대하여 반복하기를 원할 때 유용하다:

Example

Change the background color of all <p> elements in a node list:

var myNodelist = document.getElementsByTagName("p");
var i;
for (i = 0; i < myNodelist.length; i++) {
    myNodelist[i].style.backgroundColor = "red";
}
Try it Yourself »

  노드목록(node list)은 배열이 아니다!

  노드목록은 배열처럼 보이나, 아니다. 노드목록에 반복하여 배열처럼 노드들을 접근할 수있다. 하지만,  노드목록에 대하여 valueOf() 또는 join() 같은 Array 메서드를 사용할 수는 없다.
