JavaScript 문자열(String) 연산자

 + 연산자는 문자열 변수 또는 텍스트 값을 추가 하는데도 사용될 수 있다.

Example

txt1 = "John";
txt2 = "Doe";
txt3 = txt1 + " " + txt2;

The result of txt3 will be:

John Doe
Try it Yourself »

문자열을 추가하는데  +=  연산자도 사용될 수 있다 :

Example

txt1 = "What a very ";
txt1 += "nice day";

The result of txt1 will be:

What a very nice day
Try it Yourself »

 문자열과 숫자 더하기

두 숫자를 더하면 합계를 반환하지만, 숫자와 문자열을 더하면 문자열을 반환합니다:

Example

x = 5 + 5;
y = "5" + 5;
z = "Hello" + 5;

The result of x, y, and z will be:

10
55
Hello5

JavaScript 비교(Comparison) 및 논리(Logical) 연산자

비교 및 논리 연산자는 true 또는 false 를 테스트 하는데 사용된다.

비교(Comparison) 연산자

비교 연산자는 변수들이나 값들 사이의 동등성이나 차이를 결정하는 논리문(logical statements)에 사용됩니다. 

 x=5 로 주어진 경우, 아래 표는 비교 연산자를 설명합니다 :

Operator Description Comparing Returns Try it
== equal to x==8 false Try it »
x==5 true Try it »
=== exactly equal to (equal value and equal type) x==="5" false Try it »
x===5 true Try it »
!=  not equal x!=8 true Try it »
!==  not equal (different value or different type) x!=="5" true Try it »
x!==5 false Try it »
>  greater than x>8 false Try it »
<  less than x<8 true Try it »
>=  greater than or equal to x>=8 false Try it »
<=  less than or equal to x<=8 true Try it »

How Can it be Used

비교 연산자(Comparison operators)는 값을 비교하고 결과에 따라 조치를 취할 조건문에서 사용할 수 있습니다 :

if (age < 18) text = "Too young";

논리(Logical) 연산자

논리 연산자는 변수들과 또는 값들 사이의 논리를 결정하기 위해 사용된다.

t x=6 과  y=3 로 주어진 경우, 아래 표는 논리 연산자를 설명합니다:

Operator Description Example
&& and (x < 10 && y > 1) is true
|| or (x==5 || y==5) is false
! not !(x==y) is true

조건(Conditional) (Ternary) 연산자

JavaScript 는 어떤 조건에 따라 변수에 값을 할당하는 조건 연산자가 포함되어 있습니다.

Syntax

variablename = (condition) ? value1:value2 

Example

var voteable = (age < 18) ? "Too young":"Old enough";
