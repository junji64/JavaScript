범위(Scope) 는 접근할 수 있는 변수들의 집합이다.

JavaScript Scope

JavaScript 에서, 객체와 함수는 또한  변수이다.

JavaScript 에서, 범위(scope)는 접근할 수 있는 변수, 객체, 함수들의  집합이다.

JavaScript 는 함수 범위를 가지며: 범위는 함수 안에서 변한다.

지역(Local) JavaScript 변수(Variables)

JavaScript 함수 안에 선언된 변수는 그 함수에 대해서 지역(LOCAL) 이 된다.

지역 변수(Local variables)는 지역 범위(local scope)를 가진다: 그들은 그 함수 안에서만 접근할 수 있다. 

Example

// code here can not use carName

function myFunction() {
    var carName = "Volvo";

    // code here can use carName

}

지역 변수는 그들의 함수 안에서만 인식될 수 있기 때문에, 같은 이름의 변수가 다른 함수에서 사용될 수 있다.

지역 변수는 함수가 시작될 때 생성되서, 함수가 종료될 때 지워진다.

전역(Global) JavaScript 변수(Variables)

함수 바깥에 선언된 변수는 전역(GLOBAL)이 된다. .

전역 변수는 전역 범위(global scope)를 가진다: 웹페이지의 모든 스크립트와 함수가 그것에 접근할 수 있다.

Example

var carName = " Volvo";

// code here can use carName

function myFunction() {

    // code here can use carName

}

자동 전역(Automatically Global)

선언되지 않은 변수에 값을 할당하면, 자동적으로 전역변수(GLOBAL variable)가 된다.

다음 코드 예는 , 함수 안에서 선언되었을 지라도, carName 을 전역변수로 선언한다. 

Example

// code here can use carName

function myFunction() {
    carName = "Volvo";

    // code here can use carName

}

JavaScript 변수의 수명

JavaScript 변수의 수명은 선언될 때 시작한다.

지역변수는 함수가 완료될 때 삭제되며, 전역변수는 페이지를 닫을 때 삭제된다.

Function 인수(Arguments)

Function 인수(arguments) (매개변수(parameters)) 는 함수 안에서 지역변수로 작동한다.

HTML에서 전역 변수

JavaScript 로, 전역 범위는 전체 JavaScript 환경이다.

HTML 에서, 전역 범위는 window 객체이다. : 모든 전역변수는 window 객체에 속한다.

Example

// code here can use window.carName

function myFunction() {
    carName = "Volvo";
}

Did You Know?

  전역변수, 또는 함수는 window 변수나 함수를  덮어쓸 수 있다.
  window 객체를 포함한 누구나 전역 변수나 함수를 덮어 쓸 수 있다.
