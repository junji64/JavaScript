JavaScript 는 컴파일이 필요없는 scripting 프로그래밍 언어이다.

JavaScript 는 HTML 페이지에 삽입 될 수 있고, 모든 웹 브라우저에서 실행될 수 있다.

JavaScript 는 배우기 쉽습니다.

이 페이지는 JavaScript 가 수행할 수 있는 몇 가지 예를 보여줍니다.

JavaScript 는 HTML 내용을 변경할 수 있다

document.getElementById() 는 가장 흔히 사용되는 HTML의 메서드(method) 중 하나입니다.

다음의 예는 HTML 요소 ( id="demo" 인) 를 찾는데 이 메서드를 사용하고, 요소의 내용(innerHTML)을 "Hello JavaScript"로 바꾸고 있다.

Example

document.getElementById("demo").innerHTML = "Hello JavaScript";
Try it Yourself »

JavaScript 는 HTML 속성 값을 변경할 수 있다

이 예제에서는 HTML <img> 태그의  src 속성을 변경하여, HTML image를 바꾼다. :

The Light bulb

Try it Yourself »

JavaScript 는 HTML 스타일(CSS) 을 변경할 수 있다

HTML 요소의 스타일 변경은  HTML 속성 변경의 한 변형으로 볼 수 있다.

Example

document.getElementById("demo").style.fontSize = "35px";
Try it Yourself »

JavaScript 는 HTML 요소를 숨길 수 있습니다.

display 스타일을 변경하여 HTML 요소를 숨길 수 있습니다.

Example

document.getElementById("demo").style.display = "none";
Try it Yourself »

JavaScript 는 HTML 요소를 보일 수 있습니다.

display 스타일을 변경하여 숨겨진 HTML 요소를 표시 할 수도 있습니다.

Example

document.getElementById("demo").style.display = "block";
Try it Yourself »

Did You Know?

JavaScript와 Java는 개념과 디자인면에서 완전히 다른 언어입니다.
JavaScript는 1995년 Brendan Eich에 의해 발명되었으며 1997년 ECMA 표준이되었습니다.
ECMA-262는 표준의 공식 명칭입니다. ECMAScript는 언어의 공식 이름입니다..
