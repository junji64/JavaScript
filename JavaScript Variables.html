대수학(Algebra)과 유사

다음의 예에서, price1, price2, 와 total은 변수들이다.

Example

var price1 = 5;
var price2 = 6;
var total = price1 + price2;
Try it Yourself »

대수학(algebra)에서와 마찬가지로, 프로그래밍에서 우리는 값을 담는데 변수(price1 같은)를 사용한다.

대수학(algebra)에서와 마찬가지로, 프로그래밍에서 우리는 수식(total = price1 + price1)에서 변수를 사용한다.

위의 예에서, total 이 11로 계산된다.

JavaScript 식별자(Identifiers)

모든 JavaScript 변수(variable)는 고유한 이름으로 식별되어야 한다.

이러한 고유한 이름들을 식별자(identifiers)라고 한다.

식별자는 짧은 이름이거나(x 나 y 같이),  좀 더 설명적인 이름(age, sum, totalVolume 같이)일 수 있다. 

변수(고유한 식별자)의 이름 작성을 위한 일반적 규칙들은 다음과 같다:

  • 이름에는 문자, 숫자, 밑줄 과 달러 기호가 올 수 있다.
  • 이름은 문자, 밑줄,(_) 또는 달러 기호 ($)로도 시작할 수 있다.
  • 이름은 대소문자를 구분한다.
  • 예약어(JavaScript kewords) 들은 이름으로 사용될 수 없다.
   JavaScript 식별자들은 대소문자구분(case-sensitive)한다.

할당(Assignment) 연산자

JavaScript 에서, 등호(=)는 "할당" 연산자이다. ("~과 같다" 연산자가 아니라)

이는 대수학과 다르다. 다음은 대수학에서는 옳지 않다:

x = x + 5

그러나, JavaScript 에서는  허용이 되며: x + 5 의 값을 x 에 할당한다.

( x + 5 의 값을 계산하고, 그 결과를 x에 넣는다. x 의 값은 5 만큼 증가한다.)

대수와 마찬가지로, JavaScript 의 변수는 값 (x=5) 또는 식 (z=x+y)을 담을 수 있습니다.

   "~과 같다"는 연산자는 JavaScript 에서는 == 로 쓴다.

JavaScript 데이터 형식

JavaScript 변수는 100 같은 숫자와 "John Doe" 같은 텍스트 값(person="John Doe")을 담을 수 있다.

프로그래밍에서  "John Doe" 와 같은 텍스트는 문자열(string)이라고 부른다.

JavaScript  는 데의터의 많은 유형을 다룰 수 있으나, 지금은 단지 숫자(numbers) 와 문자열(strings) 만을 생각합니다.

문자열(string)은 , 큰 따옴표나 작은 따옴표 안에 넣습니다.  숫자(number)는 따옴표 없이 씁니다.

숫자 값을 따옴표로 묶을 경우, 텍스트 문자열(string)로 간주됩니다.

Example

var pi = 3.14;
var person = "John Doe";
var answer = 'Yes I am!';

JavaScript 변수 선언(생성)

JavaScript 에서 변수를 만드는 것을, 보통 변수 "선언(declaring)" 이라고 부른다.

JavaScript 의 변수는 var 키워드를 사용해서 선언한다. 

var carName;

선언 후, 변수는 (값이 없어) 비어 있습니다.

변수에 값을 지정하려면 등호를 사용합니다:

carName = "Volvo";

그러나, 선언할 때  변수에 값을 할당 할 수도 있습니다:

var carName = "Volvo";

아래의 예에서, 우리는 carname 이라는 변수를 선언하 후, 여기에 값 "Volvo"를 할당하고, id="demo" 인 HTML paragraph 의 값에 넣어 준다. :

Example

<p id="demo"></p>

<script>
var carName = "Volvo";
document.getElementById("demo").innerHTML = carName;
</script>
Try it Yourself »
It's a good programming practice to declare all the variables at the beginning of a script.

한 문장, 많은 변수

하나의 문장(statement)에서 여러 변수를 선언 할 수 있습니다. 그냥 문장을 var 로 시작하고 변수들을 쉼표 (,)로 분리한다.:

var person = "John Doe", carName = "Volvo", price = 200;
Try it Yourself »

선언이 여러 줄에 걸쳐 있을 수 있습니다 :

var person = "John Doe",
carName = "Volvo",
price = 200;
Try it Yourself »

Value = undefined

컴퓨터 프로그램에서 변수는 종종 값 없이 선언됩니다. 이 때, 값은 계산되는 것이거나, 또는 사용자 입력과 같이 나중에 제공 될 것이다.

값 없이 선언 된 변수는 undefined 값을 갖는다.

다음과 같은 명령문의 실행 후에 변수 carName 는  undefined 값이 됩니다 :

Example

var carName;

JavaScript 산술(Arithmetic)

대수학에서와 마찬가지로,  = 과 + 같은 연산자를 사용하여, JavaScript 변수에 연산을 수행할 수 있다:

Example

var x = 5 + 2 + 3;
Try it Yourself »

Example

var x = "John" + " " + "Doe";

Example
