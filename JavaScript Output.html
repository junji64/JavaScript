JavaScript 는 내장된 프린트(print) 또는 표시(display) 기능을 가지고 있지 않습니다.

JavaScript Display Possibilities

JavaScript 는 여러방법으로 데이터를 표시(display) 할 수 있다:

  • window.alert() 을 사용한 경고박스(alert box)로 출력.
  • document.write() 를 사용한 HTML 로 출력
  • innerHTML 을 사용한 HTML 요소로 출력
  • console.log() 를 사용한 브라우져 콘솔(browser console)로 출력

window.alert() 사용

데이터를표시하는데 경고박스(alert box)를 사용할 수 있다:

Example

<!DOCTYPE html>
<html>
<body>

<h1>My First Web Page</h1>
<p>My first paragraph.</p>

<script>
window.alert(5 + 6);
</script>

</body>
</html>

document.write() 사용

테스팅 목적으로, document.write()를 사용하면 편리하다.

Example

<!DOCTYPE html>
<html>
<body>

<h1>My First Web Page</h1>
<p>My first paragraph.</p>

<script>
document.write(5 + 6);
</script>

</body>
</html>
Try it Yourself »

HTML 문서가 로드된 뒤에 document.write()를 실행하면, 모든 기존의 HTML을 지우게 된다.

Example

<!DOCTYPE html>
<html>
<body>

<h1>My First Web Page</h1>
<p>My first paragraph.</p>

<button onclick="document.write(5 + 6)">Try it</button>

</body>
</html>
Try it Yourself »
   document.write() 메서드는  테스트 용으로만 사용해야합니다.

innerHTML 사용

HTML 요소에 액세스하는데, JavaScript 는  document.getElementById(id) method를 사용할 수 있습니다.

id 속성은 HTML 요소를 식별하며, innerHTML 속 성은 HTML 내용을 정의 한다.  :

Example

<!DOCTYPE html>
<html>
<body>

<h1>My First Web Page</h1>
<p>My First Paragraph</p>

<p id="demo"></p>

<script>
document.getElementById("demo").innerHTML = 5 + 6;
</script>

</body>
</html>
Try it Yourself »

colsole.log() 사용

브라우저에서 데이터를 표시하는 데 console.log() 메 소드를 사용할 수 있다.

브라우저에 서 F12로 브라우져 콘솔을 활성화하고, 메뉴에서 "Console" 을 선택합니다.

Example

<!DOCTYPE html>
<html>
<body>

<h1>My First Web Page</h1>
<p>My first paragraph.</p>

<script>
console.log(5 + 6);
</script>

</body>
</html>
