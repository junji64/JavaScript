JavaScript 시간 간격(time-intervals)으로 실행될 수 있다.

이를 타이밍 이벤트(timing event)라 부른다.

JavaScript Timing Events

JavaScript 로 , 지정된 시간 간격으로 코드를 실행할 수 있으며, 이를 타이밍 이벤트(timing events)라고 합니다.

자바 스크립트에서 이벤트에 시간을 할당하기 매우 쉽습니다. 사용되는 두 가지 주요 방법이 있습니다 :

  • setInterval() - 지정된 시간 간격으로, 계속 반복해서 함수(function)를 실행
  • setTimeout() - 지정된 밀리 초(milliseconds) 대기 후, 한 번, 함수(function)를 실행

Note:  setInterval() 와 setTimeout() 는  HTML DOM Window 객체의 두 가지 메서드입니다

setInterval() Method

setInterval() 메소드는 지정된 밀리 초(milliseconds)를 기다린 후 지정된 함수를 실행하고, 주어진 시간 간격으로 함수를 계속 실행하는 것입니다 ,

Syntax

window.setInterval("javascript function",milliseconds);

window.setInterval() 메서드는 window prefix 없이 쓸 수 있습니다.

setInterval() 메서드의 첫번째 매개 변수는 함수(function)이어야 한다. .

두번째 매개변수는 각 실행 사이의 시간 간격의 길이를 나타냅니다.

Note: 1000 milliseconds = one second.

Example

Alert "hello" every 3 seconds:

setInterval(function(){alert("Hello")},3000);

Try it yourself »

아래의 예는 현재 시간을 표시합니다.  setInterval() 메소드는 디지털 시계와 같이 함수를 1 초 마다 한 번씩 실행하는 데 사용됩니다.

Example

Display the current time:

var myVar=setInterval(function(){myTimer()},1000);

function myTimer()
{
var d=new Date();
var t=d.toLocaleTimeString();
document.getElementById("demo").innerHTML=t;
}

실행 정지

clearInterval() 메서드가 setInterval() 메소드에 의한 지정된 함수의 추가적인 실행을 중지하는 데 사용됩니다.

Syntax

window.clearInterval(intervalVariable)

window.clearInterval() 메서드는 window prefix 없이 쓸 수 있습니다.

clearInterval() 메서드를 사용 할 수 있으려면,  interval 메서드를 생성할 때 전역 변수(global variable)를 사용해야 합니다 :

myVar=setInterval("javascript function",milliseconds);

그러면, clearInterval() 메서드를 호출하여 실행을 중지 할 수 있습니다.

Example

Same example as above, but we have added a "Stop time" button:

<p id="demo"></p>
<button onclick="myStopFunction()">Stop time</button>

<script>
var myVar=setInterval(function(){myTimer()},1000);
function myTimer()
{
var d=new Date();
var t=d.toLocaleTimeString();
document.getElementById("demo").innerHTML=t;
}
function myStopFunction()
{
clearInterval(myVar);
}
</script>

The setTimeout() Method

Syntax

window.setTimeout("javascript function",milliseconds);

window.setTimeout() 메서드는 window prefix 없이 쓸 수 있습니다.

setTimeout () 메서드는 지정된 밀리 초(milliseconds)를 기다린 후 지정된 함수를 실행합니다.

setTimeout() 첫 번째 인자는 함수이어야한다.

두 번째 매개 변수는 첫 번째 매개 변수를 실행하기 전에, 지금부터 기다릴 시간을 밀리 초 단위로 나타냅니다

Example

Wait 3 seconds, then alert "Hello":

setTimeout(function(){alert("Hello")},3000);

Try it yourself »

How to Stop the Execution?

clearTimeout() 메서드의 setTimeout() 메서드에 지정된 함수의 실행을 중지하는 데 사용됩니다.

Syntax

window.clearTimeout(timeoutVariable)

window.clearTimeout() 메서드는 window prefix 없이 쓸 수 있습니다.

clearTimeout() 메서드를 사용 할 수 있으려면,  timeout 메서드를 생성할 때 전역 변수(global variable)를 사용해야합니다 :

myVar=setTimeout("javascript function",milliseconds);

그러면, 함수가 아직 실행되지 않은 경우, clearTimeout() 메서드를 호출하여 실행을 중지 할 수 있습니다.

Example

Same example as above, but we have added a "Stop the alert" button:

var myVar;

function myFunction()
{
myVar=setTimeout(function(){alert("Hello")},3000);
}

function myStopFunction()
{
clearTimeout(myVar);
}
